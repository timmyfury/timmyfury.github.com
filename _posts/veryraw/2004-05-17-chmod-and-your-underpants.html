"I want a pair of boxers that say 'chmod 777' across the crotch." To enjoy this concept I must first make sure you all understand the concept. I'll give a quick rundown for the layman. Any further explaination is on you. Linux is an operating system based on unix. chmod is a shell command for the unix-like environment. It is the abbreviation of "change mode." It's used to give different users access to different things in the network environment. This command becomes very useful in web design when you start doing file manipulation and if you don't understand it you may lose your mind. There are three user groups, owner, group and world. These are relatively self explainatory and I am sure you can figure out which each one means. Then there are three settings which must be taken into consideration when creating security policies. Read, which give the right to read. Write, which give the ability to alter the file. Then there is execute, which as the name implies gives the right to execute the file to a particular user type. You then use binary to figure out the settings. The chart makes it easy. I use my fingers though. You can check the spots you want rights for and then add the numbers in those columns and voila. You have your security settings. You would run the command by typing in the chmod then the 3 digit security setting. Full rights you would type, "chmod 777." No rights you would type, "chmod 000" Back to the original, "I want a pair of boxers that say 'chmod 777' across the crotch." This would give everyone full access to your boxer shorts.
