Last week Andrea and Jeff posted their firefox extensions. I found a few good extensions that have already become indespensible, so I thought that mine might help.

  • DownThemAll - Lets me download all the media linked on a page. I certainly would never use this for anything nefarious.
  • Colorzilla - Select the colors that are used on a page.
  • ForecastFox - weather, pretty basic.
  • Numbered Tabs - If you hit ctrl+num, you will go to that tab in the browser window. This just adds numbers to the tabs.
  • Web Developer Toolbar - A must if you do any html/css. I wish there was a way yo open the cookies panel from it, but its still pretty awesome.
  • Aardvark - Builds a little on what the old web developer toolbar did. I don't know if its useful anymore. I lets you highlight sections of a page and see what elements they are part of.
  • Greasemonkey - add behaviors to your browser with user scripts
  • LiveHTTPHeaders - This is undoubtably the most important extension I use when I am developing an application. The ability to see everything that's happening is very important. This will probably become 10 times more important when I start doing Ajax stuff.

My new extensions from Andrea and Jeff are awesome.

  • Screengrab - screenshots of an html page. Great for circumventing those idiots that try to make it so you can't download from their page.
  • FireFTP - The integrated browser FTP is vital.
